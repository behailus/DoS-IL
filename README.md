# DoS-IL Overview

Operators

Comparison: 
	=  (equality)
	<  (less than)
	<= (less than or equal)
	>  (greater than)
	>= (greater or equal)
	
	|  (logical or)
	&  (logical and)
	
Separators: 
	;  end of statement
	,  List parameters and items in format
	_  variable declaration 
	:= assignment operator 
	(  begin parameter 
	)  end parameter
	{  begin block 
	}  end of block 
	-> resolve items (used as memeber access operator in C)
